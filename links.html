Terminal applications

  • tmux - Would not use a terminal emulator without it.
  • Neovim - Give a man a Vim config and he will write code for a day. Teach a man to roll his own config, and he will never write another line of code again.
  • Doom Emacs - For when you grow bored of Vim and realize you want your text editor to also be your entire operating system. Doom Emacs comes with neat defaults like Vim keybindings and Spacemacs style workflow.
